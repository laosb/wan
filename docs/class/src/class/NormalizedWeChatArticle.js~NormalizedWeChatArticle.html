    NormalizedWeChatArticle

    Extends:

    WeChatArticle → NormalizedWeChatArticle

    NormalizedWeChatArticle is for fetching & normalize a WeChat article. It extends WeChatArticle, and has an extra method NormalizedWeChatArticle#normalize.

    Just to clarify, this class does nothing more than normalize. Except you don't need to fetchAndParse before normalization. But you can still do this manually, so you get the chance to make your own normalization work before our process, with a cheerio instance from NormalizedWeChatArticle#getContent$.

    Example:

    import { NormalizedWeChatArticle } from 'wearticle'
const aboutHolaArticle = new NormalizedWeChatArticle('http://mp.weixin.qq.com/s/5NxzEg0N18v-AuOB_RmSDw')
await article.normalize()
console.log(article.content)
